YouTube Sidebar Plugin by WTG

== Description ==

Do need to display YouTube videos in your sidebar per post or page?

This plugin lets you add and manage videos in that way...

Do you want to replace the video space with ads if no videos have been added to a post?

Our plugin will help you continue to use the widget space by displaying ads by default...

= Support =
The plugin is supported by a dedicated website at [www.webtechglobal.co.uk](http://www.webtechglobal.co.uk/) and we are more than happy
to improve the plugin in any way you need.

== Installation ==

Initial upload and activation as guided by Wordpress.org standard plugin installation.

1. Upload the `youtube-sidebar` folder to the `/wp-content/plugins/` directory
1. Activate the YouTube Sidebar plugin through the 'Plugins' menu in WordPress
1. Please get to know the Settings screen and configure the plugin to your liking
