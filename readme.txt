Add YouTube videos to your sidebar for each page on your blog using the admin!

== Description ==

Allows you to paste the YouTube embed video code into any post or page when creating or editing it in the menu. Videos will appear in your sidebar only when visiting that post or page.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Ensure plugin files are in a folder named 'youtube-sidebar' i.e `/wp-content/plugins/youtube-sidebar/youtube-sidebar.php`
