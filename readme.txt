Add YouTube videos to your sidebar for each page on your blog using custom fields and a widget. 

== Description ==
Display YouTube videos on individual pages. Let them list on a page, category view or front page or switch them off. You can even let AdSense show when a post doesnt have a video.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Ensure plugin files are in a folder named 'youtube-sidebar' i.e `/wp-content/plugins/youtube-sidebar/youtube-sidebar.php`
4. Go to Appearance in your admin menu then Widgets. Drag and drop the YouTube Sidebar widget to your sidebar to activate.
5. Add your Google AdSense code for when a post has no videos or deactivate to show nothing.
6. Copy and paste the YouTube "Embed" video code into a new custom field for any post or page. Name the custom field "youtube".
